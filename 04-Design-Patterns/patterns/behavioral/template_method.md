# Template Method Design Pattern

## Definition
The Template Method pattern defines the skeleton of an algorithm in a base class, letting subclasses override specific steps without changing the overall algorithm’s structure.

## Concept
- The base class provides a template method with the general workflow.
- Subclasses implement or override the steps as needed.

## Analogy
A cooking recipe: The overall steps are fixed (prepare, cook, serve), but each chef can customize the details.

## Benefits
- Promotes code reuse and consistency.
- Allows customization of algorithm steps.
