# Memento Design Pattern

## Definition
The Memento pattern captures and externalizes an object's internal state without violating encapsulation, so the object can be restored to this state later.

## Concept
- The originator creates a memento containing a snapshot of its state.
- The caretaker holds the memento but never modifies it.
- The originator can restore its state from the memento.

## Analogy
A game save point lets you return to a previous state of the game.

## Benefits
- Enables undo/redo functionality.
- Preserves encapsulation boundaries.
