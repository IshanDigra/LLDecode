# Flyweight Design Pattern

## Definition
The Flyweight pattern shares common parts of objects to reduce memory usage, especially when many objects are similar.

## Analogy
A library sharing one copy of each book title among many readers.

## Benefits
- Reduces memory consumption by sharing data.
- Improves performance in large-scale systems.

## Example Use Cases
- Text rendering that reuses glyph objects for repeated characters.
- Caching and sharing configuration data.
